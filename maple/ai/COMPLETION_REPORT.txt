1. PANEL (面板)
   .ui-panel
   ├─ .ui-panel--fullwidth
   ├─ .ui-panel--fixed
   └─ .ui-panel--centered

   用於: 所有遊戲 UI 容器（背包、商店、角色等）

2. TABS (分頁)
   .ui-tabs → .ui-tab.active

   用於: 面板內容切換

3. BUTTONS (按鈕)
   .btn
   ├─ .btn--action
   ├─ .btn--black
   ├─ .btn--small
   ├─ .btn--large
   └─ .btn--disabled

   用於: 所有交互操作

4. GRID (網格)
   .ui-grid
   ├─ .ui-grid--fixed (16 列)
   ├─ .ui-grid--4col
   └─ .ui-grid--fullwidth

   用於: 多個格子的容器

5. SLOT (格子)
   .ui-slot
   ├─ .active / .selected
   ├─ .empty
   ├─ .disabled
   └─ .ui-slot__badge

   用於: 單一物品/位置

6. STATUSBAR (狀態條)
   .ui-status-bar
   ├─ .ui-status-bar__icon--gold
   └─ .ui-status-bar__icon--point

   用於: 金幣、積分顯示

7. SEARCHBAR (搜尋)
   .ui-search-bar + .ui-search-input

   用於: 過濾和搜尋
